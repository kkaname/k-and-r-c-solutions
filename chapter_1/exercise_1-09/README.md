# Exercise 1.09: Replace Blank

Solution to Exercise 1.09 from *The C Programming Language* (K&R).

**Question:** Write a program to copy its input to its output, replacing each string of one or more blanks by a single blank. 

## Build & Run

### Compile:
```sh
gcc -o replace_blanks replace_blanks.c
```

### Run:
```sh
./replace_blanks
```

## Expected Output:
### Input:
```
hello              world
```
### Output:
```
hello world
```

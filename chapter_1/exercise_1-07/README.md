# Exercise 1.07: Value of EOF

Solution to Exercise 1.07 from *The C Programming Language* (K&R).

**Question:** Write a program to print the value of EOF.

## Build & Run

### Compile:
```sh
gcc -o value_eof value_eof.c
```

### Run:
```sh
./value_eof
```

## Expected Output:
```
Value of EOF: -1.
```

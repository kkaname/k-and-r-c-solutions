# Exercise 1.03: Fahrenheit to Celsius conversion table

Solution to Exercise 1.03 from *The C Programming Language* (K&R).

**Question:** Modify the temperature conversion program to print a heading above the table.

## Build & Run

### Compile:
```sh
gcc -o fahr_cels fahr_cels.c
```

### Run:
```sh
./fahr_cels
```

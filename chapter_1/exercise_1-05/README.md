# Exercise 1.05: Reverse Tempurature Conversion table

Solution to Exercise 1.05 from *The C Programming Language* (K&R).

**Question:** Modify the temperature conversion program to print the table in reverse order, that is, from 300 degrees to 0.

## Build & Run

### Compile:
```sh
gcc -o reverse_temp_table reverse_temp_table.c
```

### Run:
```sh
./reverse_temp_table
```

## Expected Output:
```
Fahrenheit    Celsius
---------------------
       300     148.89
       280     137.78
       260     126.67
       240     115.56
       220     104.44
       200      93.33
       180      82.22
       160      71.11
       140      60.00
       120      48.89
       100      37.78
        80      26.67
        60      15.56
        40       4.44
        20      -6.67
         0     -17.78
---------------------
```

# Exercise 1.06: Verify Expression

Solution to Exercise 1.06 from *The C Programming Language* (K&R).

**Question:** Verify that the expression getchar() != EOF is 0 or 1. 

## Build & Run

### Compile:
```sh
gcc -o verify_expr verify_expr.c
```

### Run:
```sh
./verify_expr
```

## Expected Output:
### Input:
```
5
Expression getchar() != EOF gives: 1.
```

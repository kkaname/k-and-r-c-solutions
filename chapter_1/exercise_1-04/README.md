# Exercise 1.04: Celsius to Fahrenheit conversion table.

Solution to Exercise 1.04 from *The C Programming Language* (K&R).

**Question:** Write a program to print the corresponding celsius to fahrenheit table.

## Build & Run

### Compile:
```sh
gcc -o cels_fahr cels_fahr.c
```

### Run:
```sh
./cels_fahr
```

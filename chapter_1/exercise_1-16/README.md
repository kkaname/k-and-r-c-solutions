# Exercise 1.16: Print longest line

Solution to Exercise 1.16 from *The C Programming Language* (K&R).

**Question:** Revise the longest_line routine of the longest-line program so it will correctly print the length of arbitrary long input lines, and as much as possible of the text. 

## Build & Run

### Compile:
```sh
gcc -o longest_line longest_line.c
```

### Run:
```sh
./longest_line
```

## Expected Output:
### Input:
```
hello world
my name is kaname
konichiwa
```
### Output:
```
Longest line (with 18 characters):
my name is kaname
```

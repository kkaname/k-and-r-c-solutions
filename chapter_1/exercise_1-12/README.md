# Exercise 1.12: Hello WOrld

Solution to Exercise 1.12 from *The C Programming Language* (K&R).

**Question:** Write a program that prints its input one word per line. 

## Build & Run

### Compile:
```sh
gcc -o word_per_line word_per_line.c
```

### Run:
```sh
./word_per_line
```

## Expected Output:
### Input:
```
Dog is a animal.
```
### Output:
```
Dog
is
a
animal.
```

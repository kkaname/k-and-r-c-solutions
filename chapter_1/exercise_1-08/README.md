# Exercise 1.08: Count Blank Characters.

Solution to Exercise 1.08 from *The C Programming Language* (K&R).

**Question:** Write a program to count blanks, tabs, and newlines. 

## Build & Run

### Compile:
```sh
gcc -o count_blank_char count_blank_char.c
```

### Run:
```sh
./count_blank_char
```

## Expected Output:
### Input:
```
hello world
how are you

```
### Output:
```
Blanks  : 3
Tabs    : 0
Newlines: 2
```
